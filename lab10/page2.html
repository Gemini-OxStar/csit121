What is WOFF?

The Web Open Font Format (WOFF) is a web font format Mozilla developed in concert with other organizartions like Type Supply. It's a compressed version of TrueType and OpenType fonts, which are used to display text on the web.

Two Versions

Two version exist: WOFF and WOFF2. The only noteworthy difference between the two is the compression algorithm used.

Using WOFF

Use the @font-face CSS property to use WOFF fonts in your HTMLs. Your content will download more efficiently.
